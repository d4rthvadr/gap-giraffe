# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-12-05

### Added
- **Application Tracker**: Complete Kanban-style tracker for job applications.
- **AI Job Analysis**: Integration with Gemini 2.5 Flash for job description analysis.
- **Resume Manager**: Support for uploading and parsing multiple resume versions.
- **Results Page**: Match scores, missing skills, and optimization suggestions.
- **Analytics Dashboard**: Visual funnel and success metrics.
- **Export**: Ability to export all application data to CSV.
- **Deployment**: Automatic build and release pipeline via GitHub Actions.

### Changed
- **UI Redesign**: Complete overhaul of the interface to a minimal, professional aesthetic (Linear/GitHub style).
- **Icons**: Replaced all emojis with Lucide SVG icons.
- **Database**: Migrated to a robust IndexedDB schema for application tracking.

### Fixed
- **AI Configuration**: robust handling of API keys and configuration loading.
